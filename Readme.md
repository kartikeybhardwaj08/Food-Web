# 🔥 Ember & Spice — Food Website

A beautifully designed food delivery website built with pure **HTML, CSS, and JavaScript**. No frameworks, no libraries — just clean code.

---

## 📁 Project Structure

```
ember-and-spice/
│
├── cart.html        → Cart Page
├── favicon-es.png   → Ember&Spice logo
├── index.html       → Main HTML file
├── style.css        → All styles + responsive design
└── README.md        → You are here!
```

---

## ✨ Features

- 🎨 **Dark warm theme** — rich brown & orange color palette
- 🖱️ **Custom cursor** — orange dot follows your mouse (desktop only)
- 🍔 **Hamburger menu** — smooth animated nav for mobile
- 📱 **Fully responsive** — works on mobile, tablet & desktop
- 🃏 **Food cards** — hover animations with dish details
- 🏷️ **Category pills** — filter by craving
- 🎁 **Promo banner** — offer code section
- ⚡ **Smooth animations** — buttons, cards, hover effects

## 📱 Responsive Breakpoints

| Breakpoint | Screen Size | Changes |
|------------|-------------|---------|
| Desktop | `> 1024px` | Full layout, side-by-side hero |
| Tablet | `≤ 1024px` | Hero stacks vertically |
| Mobile | `≤ 768px` | Hamburger menu, single column cards |
| Small Mobile | `≤ 480px` | Smaller fonts, stacked buttons |

## 🚀 How to Run

1. Download all 3 files — `index.html`, `style.css`, `script.js`
2. Keep all files in the **same folder**
3. Open `index.html` in any browser
4. That's it! No setup needed ✅
